Hoe begin ik een HTML/CSS bestand met een macbook? Wat is de basis?

1. Mapje maken

Maak in finder een mapje aan met een duidelijke naam.
Voorbeeld:
Maak een mapje aan in finder en noem het les-06.
Zorg dat je mapje ergens georganiseerd staat om het makkelijker terug te vinden.

2. Bestand openen in Visual Studio Code

Na het openen van Visual Studio Code, klik je op "open folder". Je komt nu in je finder terecht. Hier selecteer je je mapje wat je net had gemaakt.("les-04")
Dit mapje maak je nu open.

3. HTML bestand aanmaken

Nu je het mapje hebt geopend druk je op het icoontje met een blad en een plus teken.
Dit wordt je HTML bestand en dit benoem je "index.html".

4. Inhoud HTML

Nadat je een html bestand hebt aangemaakt, ga je er nu in werken.
Allereerst schrijf je op: < ! doctype html > .
Daarna begin je de basis op te schrijven. (zie foto hiernaast.)
In je body komt alles wat je op je pagina wilt hebben.
1 body = 1 pagina.

5. CSS bestand koppelen

Maak een nieuwe bestand aan, maar nu met de naam "style.css"
Wanneer je dat hebt aangemaakt ga je terug naar je html bestand.
Onder je title schrijf je nu < link rel = " stylesheet " href = " style.css " >.
Hiermee heb je nu je CSS bestand gekoppeld aan je HTML bestand.
De codes moeten allemaal zonder spaties zoals in de foto's.

6. CSS bestand inhoud

In je css bestand kan je nu de opmaak maken van je pagina.
Je kan via je id, class of via de html termen per stukje je opmaak veranderen.
In je css bestand maak je gebruik van deze icoontjes: #, {, } en ;.
